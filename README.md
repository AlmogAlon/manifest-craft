# Manifest Craft

library for creating manifests through a simple interface

- gin as the web framework
- air for live reload
- gorm
- postgres


## Requirements
- Go 1.21
- Docker
- docker-compose


## Running the application
To run the services using docker-compose:
- execute these commands: 

```bash
  cd manifest-craft
  docker-compose build
  docker-compose up
```
